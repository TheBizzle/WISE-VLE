Instructions (Multiple Choice Assessment Step)

Single Correct Answer


  • Review the question, select ONE answer radio button, then click Check Answer You'll receive feedback about the correctness/incorrectness of your choice.
  • If you didn't find the correct answer click Try Again to make another guess.

Multiple Correct Answers


  • Review the question, select TWO OR MORE check boxes, then click Check Answers You'll receive feedback about the correctness/incorrectness of your choices.
  • Note that you correctly find all of the correct answers to complete the question.
  • If you didn't find the correct answer click Try Again to make another guess.
