Is There a Better Way to Die? (Brainstorm Discussion)

Compare the following two solutions for the problem above—changing the mood to "dead" if the hunger is over 12—sleep method:

Version A
public void sleep() {
   hunger = hunger + 1;
   face.setMessage("zzzzzzzzzz");
   face.setImage("asleep");
   if (hunger > 12) {
      face.setMessage("I got too hungry!");
      face.setImage("dead");
   }
}
 		Version B
public void sleep() {
   hunger = hunger + 1;
   if (hunger > 12) {
      face.setMessage("I got too hungry!");
      face.setImage("dead");
   } else {
      face.setMessage("zzzzzzzzzz");
      face.setImage("asleep");
   }
}

Will these two versions behave identically, from the standpoint of someone playing with the toy? If not, how will the difference show up, and which version do you think is better? Why?
