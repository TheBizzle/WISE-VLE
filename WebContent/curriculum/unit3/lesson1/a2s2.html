Let's use BlueJ to open Cam's first attempt at writing a Virtual Pet toy, and take a look at what she did.

Open this project in BlueJ: "cam-v1" (in BlueJ, go to Tools>Open Current BlueJ Project to have BlueJ automatically open this project for you).

As you can see, there is a single class, named VirtualPet. Each object of this class represents a virtual pet toy.

Make a VirtualPet object now, using the object bench in the usual way. You can name it anything you like.

A window will open, containing an image that represents the mood of the Virtual Pet you just made. Cam has programmed the Virtual Pet to show a "normal" mood at the start.

The virtual pet has behaviors that you can invoke in the usual way.     
You'll see the methods "feed", "exercise", and "sleep".

Invoke all three methods on your VirtualPet object. See what happens!
